WEAR THE RISE — Standalone Website
===================================

This is a complete static website (HTML + CSS + JavaScript) for
WEAR THE RISE, a Bosele Limited Holdings brand based in Maun, Botswana.

WHAT'S INSIDE
-------------
  home.html         — Homepage (open this first)
  shop.html         — Product grid with filters & cart
  collections.html  — The four signature collections
  lookbook.html     — Editorial styling guide
  about.html        — Brand story & founders
  community.html    — Testimonials & Rise Stories
  contact.html      — Order, contact form, size guide

  styles.css        — All styling (colors, fonts, layout)
  script.js         — Cart, filters, animations, WhatsApp checkout
  images/           — Photography (heroes, products, lookbook, founders)
  _partials.html    — Reference snippets (header/footer/cart) for editing

HOW TO USE
----------
1. Unzip the folder anywhere on your computer.
2. Double-click "home.html" — it opens in your browser. Done.
3. To edit a page, right-click it -> Open with -> Notepad++.
4. To change colors, fonts, or sizes, edit "styles.css" in Notepad++
   (the brand color palette is at the top under ":root").
5. To change cart behavior or the WhatsApp number, edit "script.js"
   (your WhatsApp number sits at the very top of the file).

PUBLISHING THE SITE ONLINE
--------------------------
Upload the entire folder to any web host (cPanel, Netlify, GitHub Pages,
or your domain provider's file manager). No build step. No installation.
The folder works exactly the same online as it does on your computer.
